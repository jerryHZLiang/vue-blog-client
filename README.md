# 多人共享博客
> A Vue.js project


## 项目功能
本项目是一款多人共享博客，包含首页、用户文章列表、登录、注册、个人管理、编辑、发布等功能。
使用 Vue.js 技术栈：vue-cli/vue/axios/vue-router/vuex/es6/npm/less
